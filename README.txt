Instrucciones:

	Esta aplicación web simula un programa nativo para un teléfono inteligente.
	Para iniciar esta aplicación, abra con un explorador web el archivo "home.html".
	Esta librería se consigue en "https://fullcalendar.io/"

Librerías Usadas:

	Para esta entrega se usaron tres librerías externas. La primera es "fullcalendar-3.0.1"
	para mostrar el calendario. La vista del calendario es una modificación de uno de los 
	ejemplos que trae esta librería consigo.

	Adicionalmente, se usaron las librerías "exporting.js" y "highstock.js" para mostrar
	las gráficas en la vista de consumo eléctrico y en la vista de consumo de agua.
	Estas librerías se consiguen en "http://www.highcharts.com/"
